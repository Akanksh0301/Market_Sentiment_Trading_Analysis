# Market Sentiment vs Trading Performance Analysis

## Overview
This project analyzes how Bitcoin market sentiment affects trader performance.

## Tools Used
- Python
- Pandas
- Matplotlib
- 
## Dataset

The datasets used in this project:

- Bitcoin Fear & Greed Index  
- Historical trading data (Hyperliquid)

Due to size constraints, datasets are not included in this repository.  
They can be accessed from the original source provided in the assignment.

## Key Insights
- Highest profits during Extreme Greed
- Strong activity during Fear
- Neutral markets show weaker performance
